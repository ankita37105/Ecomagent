How It Works

Set up in 3 steps

Open Claude Code settings, paste endpoint + API key, and start coding like normal.

Step 1

Open Claude Code settings.json

Go to your local Claude Code configuration file.

Windows: %USERPROFILE%\.claude\settings.json
Step 2

Paste endpoint and API key

Use your EcomAgent endpoint and key in the env block.

"ANTHROPIC_BASE_URL": "https://api.ecomagent.in/"
"ANTHROPIC_AUTH_TOKEN": "YOUR_API_KEY"
Step 3

Use Claude Code normally

Run your usual prompts and vibe-code workflows. Nothing else changes.
