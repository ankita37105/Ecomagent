3-Step Quickstart
Integrate EcomAgent in minutes. Works directly inside Claude Code.
1

Open your settings.json

Claude Code config file on your machine.

Windows:
%USERPROFILE%\.claude\settings.json

macOS/Linux:
~/.claude/settings.json
2

Paste endpoint + API key

Add EcomAgent URL and your key in the env block.

"ANTHROPIC_BASE_URL": "https://api.ecomagent.in/"
"ANTHROPIC_AUTH_TOKEN": "YOUR_API_KEY"
3

Use Claude Code normally

Vibe-code as usual — nothing else changes.

OpenAI Python SDK
python 
from openai import OpenAI

BASE_URL = "https://api.ecomagent.in/v1"
API_KEY = "YOUR_API_KEY"


def main():
    client = OpenAI(
        api_key=API_KEY,
        base_url=BASE_URL,
    )

    response = client.chat.completions.create(
        model="claude-sonnet-4.6",
        messages=[
            {"role": "user", "content": "Hello!"}
        ],
    )

    print(response.choices[0].message.content)


if __name__ == "__main__":
    main()

Streaming with Python requests
python 
import requests

BASE_URL = "https://api.ecomagent.in/v1"
API_KEY = "YOUR_API_KEY"


def main():
    response = requests.post(
        f"{BASE_URL}/chat/completions",
        headers={
            "Authorization": f"Bearer {API_KEY}",
            "Content-Type": "application/json",
        },
        json={
            "model": "claude-opus-4.6",
            "messages": [
                {"role": "user", "content": "Hello!"}
            ],
            "stream": True,
        },
        stream=True,
    )

    for line in response.iter_lines():
        if line:
            print(line.decode("utf-8"))


if __name__ == "__main__":
    main()

Full Test Completion Script
python 
import requests
import json
import time

# Configuration
API_KEY = "YOUR_API_KEY"
BASE_URL = "https://api.ecomagent.in/v1"


def run_chat_example():
    print("\n--- Testing Model: claude-opus-4.6 ---")
    url = f"{BASE_URL}/chat/completions"
    headers = {
        "Authorization": f"Bearer {API_KEY}",
        "Content-Type": "application/json"
    }

    payload = {
        "model": "claude-opus-4.6",
        "messages": [
            {"role": "user", "content": "Hello! Are you working?"}
        ]
    }

    try:
        response = requests.post(url, headers=headers, json=payload)
        print(f"Status Code: {response.status_code}")
        if response.status_code == 200:
            print("Response Body:")
            print(json.dumps(response.json(), indent=2))
        else:
            print(f"Error: {response.text}")
    except Exception as e:
        print(f"Failed to connect: {e}")


if __name__ == "__main__":
    time.sleep(2)
    run_chat_example()

💻 Claude Code Setup

Config file location
Windows: %USERPROFILE%\.claude\settings.json macOS / Linux: ~/.claude/settings.json
Paste this into settings.json
json — settings.json 
{

    "CLAUDE_CODE_ATTRIBUTION_HEADER": "0"
  },
  "permissions": {
    "deny": [
      "WebSearch"
    ]
